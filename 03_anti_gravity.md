# 3.<ruby>北緯43度の翼<rt>アンチ・グラビティ</rt></ruby>

2026年1月、大雪に見舞われた札幌。

我々人類に自由意志が存在するのか否か。そんな深遠な問すら頭をよぎる実に示唆に満ちた結末。私は確かにその瞬間を目撃した。

ALGS Year 5 Championship。優勝カップを掲げたのは、Monsoon率いるOblivionだった。

最終局面、熾烈な弾幕の中をOblivionは誰一人欠けることなく生き延び、歓喜の瞬間を迎えた。

マッチポイントを制したOblivionは抱き合い、興奮が収まらないMonsoonのリミッターは外れ、制御不能の花火となり、オクタンの如く会場を激走した。
その模様は、全てのeスポーツの歴史の中で、間違いなく最高の瞬間であった。

超満員の観客は各々の推しチームが優勝を逃した悔しさの中、Monsoonに対しては心からの祝福を送ったことだろう。
日本推し、メイドコスプレ、彼女を一途に愛しているというキャラクターは彼の属性の一部でしかないが、愛されるには十分過ぎる。これで彼のパン屋修行の再開は、当分先になるだろう。

さて、ここからが本題である。
Apex Legends界隈には、時に熱を帯びて語られる深い溝がある。そう、「キーマウ・PAD論争」だ。

優勝したOblivionは3名全員がキーマウ（キーボード・マウス）プレイヤー。対して、いち早くマッチポイントに到達しながら準優勝に甘んじた絶対王者Team Falconsは、3名全員がPADプレイヤーである。


ここから先、PAD派の人間は読み進めないことをお勧めする。

この入力デバイスの対立は、単なる好みの問題ではない。何を欲し、何を許容するかという人間性の鋭い衝突である。

私の立場を明確にしておこう。私はキーマウ派だ。かつては違った。APEXにおいては約3年間PADを握り、エイムアシストという沼に浸かっていた。だが、私はあえてそこからキーマウへと転向した。

どちらの感覚も知っているからこそ、両者の違いが皮膚で理解できる。

PADプレイヤーが敵に照準を合わせる瞬間に感じる、あの磁石のような「吸い付き」。慣れれば、ワンマガジンで敵を屠ることは容易い。しかし、繰り返される快感の背後で、私は常に一つの疑問に苛まれていた。

「この体験は私自身のものといえるのか？」

PADプレイヤーの心理はこうだ。

勝てる方がいい → PADはキーマウより対面が強い → PADを使えば勝てる → 良いスタッツ → 自己陶酔。

客観的な視点からこの心理に添削を入れる。

勝てる方がいい（違う。とるべきは体験である） → PADはキーマウより対面が強い（その通り） → 勝てる（その通り） → 良いスタッツ（その通り） → 自己陶酔（違う。認識エラー）。

極めればキーマウが有利という説もあるが、それはRas、yukaFレベルのごく一部の天才に限る話だ。

少なくとも、競技シーンと無縁のほとんどのプレイヤーにとって「強さ」はキーマウ＜PADである。

これは常識だ。だとすれば、PADプレイヤーとは「勝って当たり前の環境の中、勝つことで喜ぶことができる人種」だということになる。

私には、PADプレイヤーが、喜びの源泉が見当たらないのに喜びを見つけてくるようで、とても奇妙に映るのだ。

キーマウプレイヤーがあえて不利なデバイスを選ぶのはなぜなのか。PADプレイヤーの目には不思議に映るらしい。彼らは我々が何を恐れているのかを知らないのだ。それを教えよう。それは、PADにワンマガで屠られることではない。

我々は、限りある時間という資源を引き換えに、不純物の混入した体験を体内に注入し、己の魂を穢すことを恐れているのだ。

エイムアシストというソフトウェアは、指先の微細なブレを補正し、敵を捉え続ける。薄められたオートエイムである。我々キーマウ派の魂はそのコードの介入に居心地の悪さを感じ忌避する。

チーターに遭遇し呪詛の暴言を吐きながら、その実、薄められたオートエイムであるアシストの恩恵は手放さない。

「チーターを憎み、己を憎まず」。

この欺瞞こそが、PAD派の正体だ。

運営が用意した公式チートを利用するもしないも個人の自由である。だが忠告しよう。その選択から、あなたの人間性が透けて見えているのだということを。

キーマウ派は肉体とデバイスが直結した純粋な「生の体験」を欲している。これはAPEXを「勝てば気持ちいいゲーム」と捉えるか、人生にとっての「エクスペリエンス」と捉えるかの違いである。

画面内に起こる純粋なゲーム体験は当然ながらキーマウに軍配が上がる。APEXほどに体験性の優れたFPSは他に存在しない。この時点で私はPAD派の感受性に疑問を持っている。

キーマウにはアシストという不自然な「重り」がない。その軽さの中で、華麗なキャラコンを決め、自らの動体視力と筋肉だけで敵を追尾しきり勝利した瞬間のカタルシスは、何物にも代えがたい。ミスは100%自分の責任。だが成功もまた、100%自分自身のものだ。

対してPADにはそのようなギャンブル性も振り幅も存在しない。

操作性のもどかしさと、あてがわれた勝利が交互にやってくるのだ。私には耐えられない。

哲学者スピノザはかつてこう説いた。

「もし、空を飛んでいる石に意識があったなら、その石はきっとこう思うだろう。『自分は今、自らの意思で飛んでいるのだ』と」

これこそが、エイムアシストという名のゆりかごの中でワンマガジンを決め、快楽を得ている者の正体だ。彼らは実際には「飛ばされている」石ころに過ぎない。

その「石」の頂点に立つチームの名が、あろうことかTeam Falconsだという事実には運命的な皮肉を感じざるを得ない。

ハヤブサの名を冠しながら、彼らは自らの翼ではなく、ソフトウェアというプログラムコードに身を委ねた。それはもはや鳥ではなく「非常によく飛ぶ石」に過ぎないのだ。

ここまで持論を述べてきたが私は知っている。この、「キーマウ・PAD論争」は決して終わらないことを。

人間は強烈な恒常性に護られた生き物だ。あなたの特性は遺伝子に刻まれている。あなたがどちら側の人間かは、基本的には選べることではない。

結局のところ、スタッツという数字の魔力に抗えない計算機になるのか。不確定な未来に魂を焼かれることを厭わない表現者になるのか。それは後天的努力では変えがたい、個々のパーソナリティという恒常性の問題に帰結する。

我々人類に自由意志はあるか否か。それは当事者である人類自身には解明できないテーマだろう。人類が単細胞生物のアメーバを顕微鏡で覗き込み、その挙動をすべて化学反応だと断定できるように、我々の自由意志の謎を解明するには、人類を外側から俯瞰する『高次元の第三者』の視点が不可欠なのだ。

それでも私は信じたい。重力に違和感を感じ、キャラコンに憧れ、数字よりも体験を選んだという事実は私の自由意志であったのだと。


最後に、Team FalconsのIGLであり、Apex Legendsの象徴ImperialHalに想いを馳せる。

かつてキーマウで世界の頂点に立ちながら、葛藤の末に翼を捨てた彼は、最強の石となった。だが、札幌の空で彼を追い越していったのは、重力に叛逆したMonsoonの、あまりに人間的な、あまりに不規則な翼だった。

気の遠くなるような努力と研鑽を経て、彼らが私には理解の及ばない境地にいることはわかっている。だが、競技において結果が全てであるという冷厳な事実からは逃れられない。

今回、Team Falconsを追う3位に輝いたE36。彼らもまた、3人全員が翼を持つチームであった。トップ3のうち2チームがキーマウであるという事実は、効率主義の敗北を静かに告げている。

最強『Team Falcons』は、これから何度でも『最強の2位』になれるだろう。だが、無機質なコードに魂へのアクセスを許している限り、頂点の景色は決して見えない。

Monsoonの激情から、魂の在り方を思い出してほしい。

ImperialHal。あなたがすべきことは、もう言うまでもないだろう。

効率が魂よりも優先される時代になった。その傾向はAIの登場により急激に加速されてゆくのだろう。それでもなお我々は石であることを拒み、不格好に羽ばたき続けることができるのだろうか。
